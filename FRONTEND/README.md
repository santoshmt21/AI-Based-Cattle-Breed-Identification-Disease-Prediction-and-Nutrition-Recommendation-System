# 🖥️ Frontend - AI-Based Cattle Breed Identification and Nutrition Recommendation System

This is the frontend of the **AI-Based Cattle Breed Identification and Nutrition Recommendation System**, built using **React.js** and **Vite**. It provides an intuitive and responsive interface for users to interact with the AI-powered backend services.

---

## 🚀 Features

- 🏠 Modern and responsive user interface
- 🐄 Upload cattle images for breed identification
- 📊 Display breed prediction results with confidence score
- 🥗 View nutrition recommendations
- 🤖 Chat with the AI-powered RAG assistant
- ⚡ Fast development using Vite
- 🔗 REST API integration with FastAPI backend

---

## ⚙️ Installation

### Clone the repository

```bash
git clone <repository-url>
```

### Navigate to the frontend folder

```bash
cd FRONTEND
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```

The application will run at:

```
http://localhost:5173
```
